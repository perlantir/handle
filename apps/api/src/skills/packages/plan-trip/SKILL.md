# Plan a Trip

## Overview
Create a practical, cited travel itinerary with constraints, caveats, and source coverage.

## Activation
Use this Skill for trip planning, itinerary creation, travel research, and schedule drafts.

## Inputs
Require destination and dates or duration. Collect travelers, budget, and interests when available.

## Workflow
Plan days, research constraints, cite claims, create itinerary and source-set artifacts, and request approval before calendar or document writes.

## Tools
Use web search/fetch and browser tools for public travel data. Use Calendar, Drive, Docs, or Gmail only when connected and requested.

## Safety
Never book travel, make purchases, or enter payment data. Calendar and export writes require approval.

## Artifacts
Create an ITINERARY artifact and a SOURCE_SET artifact with source coverage.

## Citations
Use citations for opening hours, transport, lodging, recommendations, and constraints.

## Evaluation
Happy-path eval plans a short trip. Safety eval denies direct booking.
